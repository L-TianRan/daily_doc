# Markdown里添加图片
markdown里添加图片跟添加超链接很像。
- 超链接
```
[显示名称](链接地址)
```
- 图片

插入图片就是比插入超链接多一个感叹号。
  - 使用绝对地址
  ```
  ![图片名](图片绝对地址)
  ![示例图](https://github.com/L-TianRan/daily_doc/imgs/example.png)
  ```
  - 使用相对地址
  
假设我现在编辑的文件就在`daily_doc`项目的根目录里，想要添加的图片在下面的`imgs`文件夹里。我们还能使用相对路径。
  ```
  ![图片名](图片相对路径)
  ![示例图](imgs/example.png)
  ```
